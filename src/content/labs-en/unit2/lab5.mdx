This laboratory marks the beginning of working with POSIX threads (PThreads). You will learn to:

- Create threads using the `pthread` library
- Execute functions concurrently
- Understand the basic usage of `pthread_create` and `pthread_join`

## Compilation and Execution

```bash
# Basic compilation
gcc -g -Wall -o prog00-HelloWorld prog00-HelloWorld.c -lpthread

gcc -g -Wall -o pth_hello pth_hello.c -lpthread

# Execution
./prog00-HelloWorld 4
./pth_hello 4
```
