## Compilation and Execution

### Using VSCode
1. Open the tutorial folder in VSCode
2. Open the `.cpp` file you want to run
3. Press `Ctrl+Shift+B` (Windows) or `Cmd+Shift+B` (macOS)
4. The program will compile and run automatically


<Alert type="tip">
  The system will automatically create a `build/` folder in the source file's directory, compile, and place the executable inside that folder.
</Alert>

### Manual Compilation

If you prefer to use the terminal directly:

#### On Windows
```bash
cd <example_directory>
mkdir build
g++ archivo.cpp -o build/archivo.exe && ./build/archivo.exe
```

#### On macOS
```bash
cd <example_directory>
mkdir -p build
g++-14 archivo.cpp -o build/archivo && ./build/archivo
```

<Alert type="info">
  This method may seem tedious, but it is good practice for learning how to use the terminal and understanding how the compiler works.
</Alert>
