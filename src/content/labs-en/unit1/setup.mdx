## Installing MPI on Windows

Before compiling MPI code from VSCode on Windows, you need to install MS-MPI from the official Microsoft page:

1. Go to [Microsoft MPI](https://www.microsoft.com/en-us/download/details.aspx?id=105289)
2. Click **Download** and download both installers:
   - `msmpisetup.exe` (7.4 MB) → Install this one first
   - `msmpisdk.msi` (2.2 MB) → Install this one after the previous

## Compiling and Running Your MPI Program

1. Open the `.c` file containing your MPI program
2. Press `Ctrl + Shift + B` to compile the file using VSCode tasks
3. An `.exe` file will be generated at the same level as the `.c` file
4. Open the integrated terminal and run:

<CodeBlock language="bash" content={`mpiexec -n 4 ./NombreDelArchivo.exe`} />

You are now running your first MPI program from VSCode with GCC and MS-MPI.

<Alert type="tip">
  If you have issues with paths containing spaces (such as "Program Files"), make sure to separate flags and values in `tasks.json`, like `"-I", "C:\\\\Program Files (x86)\\\\Microsoft SDKs\\\\MPI\\\\Include"`.
</Alert>
