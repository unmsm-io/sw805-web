## Compilación y Ejecución

```bash
# Para C
mpicc -g -Wall -o out <program_name>.c

# Para C++
mpicxx -g -Wall -o out <program_name>.cpp

# Ejecutar con 4 procesos
mpiexec -n 4 ./out
```
