## Compilación y Ejecución

### Usando VSCode
1. Abre la carpeta del tutorial en VSCode
2. Abre el archivo `.cpp` que deseas ejecutar
3. Presiona `Ctrl+Shift+B` (Windows) o `Cmd+Shift+B` (macOS)
4. ¡Listo! El programa se compilará y ejecutará automáticamente


<Alert type="tip">
  El sistema creará automáticamente una carpeta `build/` en el directorio del archivo fuente, compilará y colocará el ejecutable dentro de esta carpeta.
</Alert>

### Compilación Manual

Si prefieres usar la terminal directamente:

#### En Windows
```bash
cd <directorio_del_ejemplo>
mkdir build
g++ archivo.cpp -o build/archivo.exe && ./build/archivo.exe
```

#### En macOS
```bash
cd <directorio_del_ejemplo>
mkdir -p build
g++-14 archivo.cpp -o build/archivo && ./build/archivo
```

<Alert type="info">
  Este método puede parecer tedioso, pero es una buena práctica para aprender a usar la terminal y entender cómo funciona el compilador.
</Alert>
