## Introducción

Escribir código paralelo correcto es solo la mitad del trabajo. La otra mitad es saber si realmente es más rápido, cuánto más rápido, y bajo qué condiciones deja de serlo. Sin métricas precisas, las afirmaciones de rendimiento son anécdotas.

En este laboratorio aprenderás a medir tiempo de ejecución con herramientas apropiadas para CPU y GPU, a calcular speedup y eficiencia, y a construir una metodología de benchmarking reproducible. Compararás cuatro implementaciones de multiplicación de matrices —secuencial, OpenMP, Pthreads y CUDA— con datos reales.

<Alert type="info">
La medición incorrecta de tiempo es uno de los errores más comunes en trabajos de computación paralela. Este laboratorio establece las herramientas y la disciplina que usarás en todos los proyectos subsiguientes.
</Alert>

## 1. Métricas de Rendimiento

### Speedup

El **speedup** mide cuánto más rápido es la versión paralela respecto a la secuencial:

$$S(p) = \frac{T(1)}{T(p)}$$

Donde $T(1)$ es el tiempo secuencial y $T(p)$ es el tiempo con $p$ procesadores.

- $S(p) = p$ : speedup lineal (ideal, raramente alcanzable)
- $S(p) < p$ : speedup sublineal (lo habitual, por overhead y código serial)
- $S(p) > p$ : speedup superlineal (ocurre por efectos de caché, rarísimo)

### Eficiencia

La **eficiencia** normaliza el speedup por el número de procesadores:

$$E(p) = \frac{S(p)}{p} = \frac{T(1)}{p \cdot T(p)}$$

Una eficiencia de 0.85 significa que cada procesador está siendo utilizado al 85% de su capacidad potencial.

### Ley de Amdahl

La fracción serial $f$ del programa limita el speedup máximo alcanzable:

$$S_{max} = \frac{1}{f + \frac{1-f}{p}}$$

Si el 10% del código es inherentemente secuencial, el speedup máximo posible es $10\times$, sin importar cuántos procesadores uses.

### Escalabilidad

| Tipo | Definición | Cómo medirla |
|------|-----------|--------------|
| **Strong scaling** | Mismo tamaño de problema, más procesadores | Fija N, varía p = 1,2,4,8,16 |
| **Weak scaling** | Crece el problema proporcionalmente con los procesadores | N = N₀ × p, varía p |

En weak scaling ideal, el tiempo permanece constante. En strong scaling ideal, el tiempo se reduce a la mitad al doblar los procesadores.

## 2. Herramientas de Medición

Elegir la herramienta de timing correcta importa. Las herramientas incorrectas introducen error de medición mayor que la diferencia que intentas medir.

| Herramienta | Resolución | Uso |
|-------------|-----------|-----|
| `time` (shell) | ~10ms | Solo para procesos completos, nunca para microbenchmarks |
| `clock()` (C) | Variable | Mide tiempo de CPU, no tiempo de pared. Evitar. |
| `clock_gettime(CLOCK_MONOTONIC)` | ~1ns | Correcto para CPU. Usa este. |
| `std::chrono::high_resolution_clock` | ~1ns | Correcto para C++. |
| `cudaEventRecord` | ~0.5µs | El único método correcto para GPU. Mide tiempo de GPU, no CPU. |

<Alert type="warning">
Nunca uses `clock()` para medir código paralelo. Mide ciclos de CPU consumidos por el proceso, que en programas multi-hilo es la suma de todos los hilos y puede ser mayor que el tiempo real.
</Alert>

### Ejercicio 1: Framework de benchmarking

Escribe una biblioteca reutilizable de timing que usarás en el resto del laboratorio.

**`benchmark.h`** — Definición de la interfaz:

```c
#ifndef BENCHMARK_H
#define BENCHMARK_H

#include <time.h>
#include <stdio.h>

typedef struct {
    double start_ms;
    double end_ms;
} Timer;

static inline double now_ms() {
    struct timespec ts;
    clock_gettime(CLOCK_MONOTONIC, &ts);
    return ts.tv_sec * 1000.0 + ts.tv_nsec / 1.0e6;
}

static inline void timer_start(Timer *t) {
    t->start_ms = now_ms();
}

static inline void timer_stop(Timer *t) {
    t->end_ms = now_ms();
}

static inline double timer_elapsed(const Timer *t) {
    return t->end_ms - t->start_ms;
}

static inline void print_result(const char *label, double elapsed_ms,
                                 double seq_ms, int threads) {
    double speedup = seq_ms / elapsed_ms;
    double efficiency = speedup / (double)threads;
    printf("%-25s  %8.2f ms  speedup=%5.2fx  efic=%5.1f%%\n",
           label, elapsed_ms, speedup, efficiency * 100.0);
}

#endif
```

**`timer_gpu.h`** — Wrapper para eventos CUDA:

```cpp
#ifndef TIMER_GPU_H
#define TIMER_GPU_H

#include <cuda_runtime.h>
#include <stdio.h>

typedef struct {
    cudaEvent_t start;
    cudaEvent_t stop;
} GpuTimer;

static inline void gpu_timer_create(GpuTimer *t) {
    cudaEventCreate(&t->start);
    cudaEventCreate(&t->stop);
}

static inline void gpu_timer_start(GpuTimer *t) {
    cudaEventRecord(t->start);
}

static inline float gpu_timer_stop(GpuTimer *t) {
    float ms;
    cudaEventRecord(t->stop);
    cudaEventSynchronize(t->stop);
    cudaEventElapsedTime(&ms, t->start, t->stop);
    return ms;
}

static inline void gpu_timer_destroy(GpuTimer *t) {
    cudaEventDestroy(t->start);
    cudaEventDestroy(t->stop);
}

#endif
```

<Alert type="tip">
Siempre ejecuta el kernel al menos una vez como "calentamiento" antes de medir. La primera invocación puede incluir tiempos de inicialización del driver CUDA que distorsionan los resultados.
</Alert>

## 3. Ejercicio 2: Benchmark de Multiplicación de Matrices

La multiplicación de matrices C = A × B para matrices de N×N es O(N³), con acceso intensivo a memoria. Es un benchmark estándar en HPC.

Implementa las cuatro versiones en el mismo archivo `benchmark.cu`:

```cpp
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <math.h>
#include <omp.h>
#include <pthread.h>
#include <cuda_runtime.h>
#include "benchmark.h"
#include "timer_gpu.h"

#define MAX_N 2048
#define TILE_SIZE 16

typedef struct {
    float *A;
    float *B;
    float *C;
    int N;
    int row_start;
    int row_end;
} MatMulArgs;

void matmul_seq(const float *A, const float *B, float *C, int N) {
    for (int i = 0; i < N; i++)
        for (int k = 0; k < N; k++)
            for (int j = 0; j < N; j++)
                C[i * N + j] += A[i * N + k] * B[k * N + j];
}

void matmul_openmp(const float *A, const float *B, float *C, int N, int threads) {
    #pragma omp parallel for num_threads(threads) schedule(static)
    for (int i = 0; i < N; i++)
        for (int k = 0; k < N; k++)
            for (int j = 0; j < N; j++)
                C[i * N + j] += A[i * N + k] * B[k * N + j];
}

void *pthread_worker(void *arg) {
    MatMulArgs *a = (MatMulArgs *)arg;
    for (int i = a->row_start; i < a->row_end; i++)
        for (int k = 0; k < a->N; k++)
            for (int j = 0; j < a->N; j++)
                a->C[i * a->N + j] += a->A[i * a->N + k] * a->B[k * a->N + j];
    return NULL;
}

void matmul_pthreads(const float *A, const float *B, float *C, int N, int threads) {
    pthread_t *tids = malloc(threads * sizeof(pthread_t));
    MatMulArgs *args = malloc(threads * sizeof(MatMulArgs));
    int rows_per_thread = N / threads;

    for (int t = 0; t < threads; t++) {
        args[t].A = (float *)A;
        args[t].B = (float *)B;
        args[t].C = C;
        args[t].N = N;
        args[t].row_start = t * rows_per_thread;
        args[t].row_end = (t == threads - 1) ? N : args[t].row_start + rows_per_thread;
        pthread_create(&tids[t], NULL, pthread_worker, &args[t]);
    }
    for (int t = 0; t < threads; t++) pthread_join(tids[t], NULL);

    free(tids);
    free(args);
}

__global__ void matmul_cuda_kernel(const float *A, const float *B, float *C, int N) {
    __shared__ float tileA[TILE_SIZE][TILE_SIZE];
    __shared__ float tileB[TILE_SIZE][TILE_SIZE];

    int row = blockIdx.y * TILE_SIZE + threadIdx.y;
    int col = blockIdx.x * TILE_SIZE + threadIdx.x;
    float sum = 0.0f;

    for (int t = 0; t < (N + TILE_SIZE - 1) / TILE_SIZE; t++) {
        int aCol = t * TILE_SIZE + threadIdx.x;
        int bRow = t * TILE_SIZE + threadIdx.y;

        tileA[threadIdx.y][threadIdx.x] = (row < N && aCol < N) ? A[row * N + aCol] : 0.0f;
        tileB[threadIdx.y][threadIdx.x] = (bRow < N && col < N) ? B[bRow * N + col] : 0.0f;
        __syncthreads();

        for (int k = 0; k < TILE_SIZE; k++)
            sum += tileA[threadIdx.y][k] * tileB[k][threadIdx.x];
        __syncthreads();
    }

    if (row < N && col < N) C[row * N + col] = sum;
}

void matmul_cuda(const float *h_A, const float *h_B, float *h_C, int N, float *elapsed_ms) {
    int bytes = N * N * sizeof(float);
    float *d_A, *d_B, *d_C;
    cudaMalloc(&d_A, bytes);
    cudaMalloc(&d_B, bytes);
    cudaMalloc(&d_C, bytes);

    cudaMemcpy(d_A, h_A, bytes, cudaMemcpyHostToDevice);
    cudaMemcpy(d_B, h_B, bytes, cudaMemcpyHostToDevice);
    cudaMemset(d_C, 0, bytes);

    dim3 block(TILE_SIZE, TILE_SIZE);
    dim3 grid((N + TILE_SIZE - 1) / TILE_SIZE, (N + TILE_SIZE - 1) / TILE_SIZE);

    GpuTimer gt;
    gpu_timer_create(&gt);
    gpu_timer_start(&gt);
    matmul_cuda_kernel<<<grid, block>>>(d_A, d_B, d_C, N);
    *elapsed_ms = gpu_timer_stop(&gt);
    gpu_timer_destroy(&gt);

    cudaMemcpy(h_C, d_C, bytes, cudaMemcpyDeviceToHost);
    cudaFree(d_A); cudaFree(d_B); cudaFree(d_C);
}

int main() {
    int sizes[] = {256, 512, 1024, 2048};
    int num_threads = 4;

    for (int si = 0; si < 4; si++) {
        int N = sizes[si];
        int bytes = N * N * sizeof(float);
        float *A = malloc(bytes);
        float *B = malloc(bytes);
        float *C = calloc(N * N, sizeof(float));

        for (int i = 0; i < N * N; i++) {
            A[i] = (float)rand() / RAND_MAX;
            B[i] = (float)rand() / RAND_MAX;
        }

        printf("\n=== N = %d ===\n", N);

        Timer t;
        double seq_ms = 0.0;

        if (N <= 1024) {
            timer_start(&t);
            matmul_seq(A, B, C, N);
            timer_stop(&t);
            seq_ms = timer_elapsed(&t);
            printf("%-25s  %8.2f ms  speedup= 1.00x  efic=100.0%%\n", "Secuencial", seq_ms);
        } else {
            printf("Secuencial N=2048: omitido (> 30s)\n");
            seq_ms = -1;
        }

        memset(C, 0, bytes);
        timer_start(&t);
        matmul_openmp(A, B, C, N, num_threads);
        timer_stop(&t);
        double omp_ms = timer_elapsed(&t);
        if (seq_ms > 0) print_result("OpenMP (4 hilos)", omp_ms, seq_ms, num_threads);
        else printf("%-25s  %8.2f ms\n", "OpenMP (4 hilos)", omp_ms);

        memset(C, 0, bytes);
        timer_start(&t);
        matmul_pthreads(A, B, C, N, num_threads);
        timer_stop(&t);
        double pth_ms = timer_elapsed(&t);
        if (seq_ms > 0) print_result("Pthreads (4 hilos)", pth_ms, seq_ms, num_threads);
        else printf("%-25s  %8.2f ms\n", "Pthreads (4 hilos)", pth_ms);

        float cuda_ms = 0.0f;
        matmul_cuda(A, B, C, N, &cuda_ms);
        if (seq_ms > 0) print_result("CUDA (tiled)", (double)cuda_ms, seq_ms, 1);
        else printf("%-25s  %8.2f ms\n", "CUDA (tiled)", cuda_ms);

        free(A); free(B); free(C);
    }
    return 0;
}
```

## 4. Ejercicio 3: Análisis de Escalabilidad

### Strong Scaling

Fija el tamaño del problema en N=1024 y varía el número de hilos OpenMP de 1 a 16. Anota los tiempos y calcula speedup.

```c
#include <stdio.h>
#include <stdlib.h>
#include <omp.h>
#include "benchmark.h"

#define N 1024

void matmul_omp(const float *A, const float *B, float *C, int n, int p) {
    #pragma omp parallel for num_threads(p) schedule(static)
    for (int i = 0; i < n; i++)
        for (int k = 0; k < n; k++)
            for (int j = 0; j < n; j++)
                C[i * n + j] += A[i * n + k] * B[k * n + j];
}

int main() {
    int thread_counts[] = {1, 2, 4, 8, 16};
    int bytes = N * N * sizeof(float);
    float *A = malloc(bytes), *B = malloc(bytes), *C = malloc(bytes);

    for (int i = 0; i < N * N; i++) { A[i] = 1.0f; B[i] = 1.0f; }

    printf("Strong Scaling (N=%d fijo):\n", N);
    printf("%-8s  %-12s  %-10s  %-10s\n", "Hilos", "Tiempo (ms)", "Speedup", "Efic (%)");

    double t1 = 0.0;
    for (int ti = 0; ti < 5; ti++) {
        int p = thread_counts[ti];
        memset(C, 0, bytes);
        Timer t;
        timer_start(&t);
        matmul_omp(A, B, C, N, p);
        timer_stop(&t);
        double ms = timer_elapsed(&t);

        if (p == 1) t1 = ms;
        double speedup = t1 / ms;
        double eff = speedup / p * 100.0;
        printf("%-8d  %10.2f ms  %8.2fx  %8.1f%%\n", p, ms, speedup, eff);
    }

    printf("\nWeak Scaling (N crece con p, N0=%d):\n", 256);
    printf("%-8s  %-8s  %-12s  %-10s\n", "Hilos", "N", "Tiempo (ms)", "Efic (%)");

    double t_base = 0.0;
    int n_base = 256;
    for (int ti = 0; ti < 5; ti++) {
        int p = thread_counts[ti];
        int n = n_base * p;
        if (n > 2048) { printf("%-8d  N=%d demasiado grande, omitido\n", p, n); continue; }

        int b2 = n * n * sizeof(float);
        float *A2 = malloc(b2), *B2 = malloc(b2), *C2 = malloc(b2);
        for (int i = 0; i < n * n; i++) { A2[i] = 1.0f; B2[i] = 1.0f; }
        memset(C2, 0, b2);

        Timer t;
        timer_start(&t);
        matmul_omp(A2, B2, C2, n, p);
        timer_stop(&t);
        double ms = timer_elapsed(&t);

        if (p == 1) t_base = ms;
        double eff = t_base / ms * 100.0;
        printf("%-8d  %-8d  %10.2f ms  %8.1f%%\n", p, n, ms, eff);

        free(A2); free(B2); free(C2);
    }

    free(A); free(B); free(C);
    return 0;
}
```

```bash
gcc -O2 -fopenmp -o scaling scaling.c -lm
./scaling
```
