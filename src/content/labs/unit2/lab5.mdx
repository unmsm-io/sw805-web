Este laboratorio marca el inicio del trabajo con hilos POSIX (PThreads). Aprenderás a:

- Crear hilos usando la librería `pthread`
- Ejecutar funciones concurrentemente
- Comprender el uso básico de `pthread_create` y `pthread_join`

## Compilación y Ejecución

```bash
# Compilación básica
gcc -g -Wall -o prog00-HelloWorld prog00-HelloWorld.c -lpthread

gcc -g -Wall -o pth_hello pth_hello.c -lpthread

# Ejecución
./prog00-HelloWorld 4
./pth_hello 4
```
